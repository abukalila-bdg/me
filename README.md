# Abu Kalila - Personal Website

Welcome to the repository for my personal website. This site serves as a professional portfolio, showcasing my experience, skills, and projects as a Senior Software Engineer and Leader.

## Overview

- **Name**: Abu Kalila
- **Role**: Senior Software Engineer & Engineering Leader
- **Focus**: Backend Systems, Team Leadership, Digital Solutions for UMKM

## Features

- **Monochrome Design**: A clean, professional, and modern aesthetic using a grayscale palette.
- **Responsive Layout**: Optimized for desktop, tablet, and mobile devices.
- **Sections**:
  - **About**: Introduction and professional summary.
  - **Experience**: Detailed career highlights and technical expertise.
  - **GitHub**: Showcase of open-source repositories and contributions.
  - **Contact**: Easy ways to get in touch via WhatsApp or Email.

## Tech Stack

- **HTML5**: Semantic structure.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Google Fonts**: 'Inter' font for typography.
- **DiceBear**: For avatar generation (placeholder).

## Setup

This is a static website. You can view it by simply opening `index.html` in any modern web browser.

No build process is required as it uses the CDN version of Tailwind CSS for simplicity during development.
